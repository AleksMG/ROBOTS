Документация API робота

Основные свойства

robot.x, robot.y - текущие координаты робота

robot.angle - текущий угол поворота (0-359)

robot.health - текущее здоровье (0-100)

robot.energy - текущая энергия (0-100)

robot.weapon - текущее оружие ('pistol', 'laser', 'rocket')

Движение

robot.move(angle) - двигаться в указанном направлении (0-359)

robot.moveTo(x, y) - двигаться к указанным координатам

robot.rotateTo(x, y) - повернуться к указанным координатам

robot.rotate(angle) - повернуться на указанный угол

robot.stop() - остановиться

Боевые действия

robot.fire() - выстрелить в текущем направлении

robot.setWeapon('weapon') - сменить оружие

robot.canSee(x, y) - проверяет, видит ли робот указанную точку

robot.scan() - возвращает расстояние до ближайшего объекта

Окружение

enemy - объект вражеского робота (имеет те же свойства)

arena.width, arena.height - размеры арены

arena.getTime() - возвращает текущее время боя (мс)
